Workflow and service examples

This site contains examples of how back end cheminformatics services can be tied into workflows. Currently we have 2 simple demos:

  1. A set of File based demos (plus a description of how these are implemented.
  2. A "work in progress" set of drag-and-drop examples (note that the workflow building part is not yet functional). In these examples structure files are uploaded to the server and stored there so that once uploaded all processing takes place on the server. These examples used ChemAxon's Marvin and JChem, Chemistry Development Kit and RDKit.

IMPORTANT: these examples are provided to indicate the possibilities of what you can do with our technology. They are not expected to be used for real work, and if you do want to implement things using these approaches your will need the necessary licenses for the underlying technology. For more information contact us info(at)informaticsmatters(dot)com.

WARNING: when trying out these examples examples you should use relatively small input files as file uploads are involved and the server on which it is running is very small. Please limit the files to hundreds or at most a few thousand molecules.
SDF or smiles formats should be OK. Results are returned in SDF format.
